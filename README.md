# 🎮 Mochi-Renew

自动续期 [Mochi Hosting](https://hosting.aida0710.work) Minecraft 服务器。

---

## 🔐 Secrets 配置

| 变量 | 格式 | 说明 |
|------|------|------|
| 🎮 `MOCHI_ACCOUNT` | `服务器名,邮箱,密码` | 登录账号 |
| 📢 `TG_BOT` | `chat_id,bot_token` | Telegram 通知 |
| 🔑 `PRIVATE_REPO_TOKEN` | `ghp_xxxxxx` | 私库读取权限 PAT |
| 🌐 `NODE_LINK` | `hysteria2://...` | sing-box 代理节点 |

---

## 📝 示例

```
MOCHI_ACCOUNT=Mochi-JP 🇯🇵,user@example.com,password123
TG_BOT=123456789,bot_token_here
```
